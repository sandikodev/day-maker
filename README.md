## How to Contribute

### Sugestion
1. use bun or pnpm instead of npm or yarn

### Contribute
1. fork this repo
2. clone your's
3. make a change
4. push and pull-request

### Develop
```
git clone this-repos.git
cd day-maker
bun install
bun dev
```

### Build
```
remix build
-> build/
-> public/build/
```

## Tech Stack
1. styling: tailwind
2. component: shadcn
3. orm: supabase
